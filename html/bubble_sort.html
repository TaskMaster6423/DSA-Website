Bubble Sort Logic

Repeatedly swaps adjacent elements if they are in the wrong order.

  1. Compare current element with the next one.
  2. If Left > Right, trigger a Swap (Slide positions).
  3. Move to the next pair.
  4. Repeat until the largest element bubbles to the end.
void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        for (int j = 0; j < n - i - 1; j++) {
            if (arr[j] > arr[j + 1]) {
                // Swap
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}
