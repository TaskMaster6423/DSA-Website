Selection Sort Logic

Splits the list into sorted (left) and unsorted (right) parts.

  1. Assume the first unsorted element is the Minimum.
  2. Scan the rest of the array. If you find a smaller number, update the Minimum.
  3. At the end of the scan, Swap the found Minimum with the first unsorted element.
  4. Repeat until the array is sorted.
void selectionSort(int arr[], int n) {
    for (int i = 0; i < n - 1; i++) {
        int min_idx = i;
        for (int j = i + 1; j < n; j++) {
            if (arr[j] < arr[min_idx])
                min_idx = j;
        }
        // Swap the found minimum element with the first element
        int temp = arr[min_idx];
        arr[min_idx] = arr[i];
        arr[i] = temp;
    }
}
