Algorithm Logic

  1. Start from the very first element (index 0) of the list.
  2. Compare the current element with the Target value.
  3. If the current element matches the target, Stop. You found it!
  4. If it doesn't match, move to the next element.
  5. Repeat until a match is found or the list ends.

int linearSearch(int arr[], int n, int x) {
    for (int i = 0; i < n; i++) {
        if (arr[i] == x)
            return i; // Element found at index i
    }
    return -1; // Element not found
}
