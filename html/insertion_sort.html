Insertion Sort Logic

  1. Pick: Lift the element at index i (Key).
  2. Scan: Look at elements to the left (index j).
  3. Shift: If arr[j] > Key, slide arr[j] to the right to make space.
  4. Insert: When a smaller element is found (or start of array), drop the Key into the empty slot.
void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j]; // Shift Right
            j = j - 1;
        }
        arr[j + 1] = key; // Insert Key
    }
}
